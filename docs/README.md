## 规范

- 文件命名：`NN-kebab-name.md`（两位编号 + 小写英文 + 横杠分隔）
- 编号用于排序，不代表优先级
- 每篇文档必含：设计细节 + 代码引用（文件路径） + 原子化提交计划 + 6DQ 质量计划
- 文档不含：工作量评估
- 本 README 实时维护下方索引
